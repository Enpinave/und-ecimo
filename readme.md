## 🛠️ Requisitos

Antes de ejecutar el proyecto, asegúrate de tener lo siguiente instalado:

- ✅ Python 3.10 o superior  
- ✅ Entorno virtual configurado (`venv`)  
- ✅ Flet instalado:
  ```bash
  pip install flet
  ```

---

## 🚀 Cómo ejecutar el proyecto

1. **Activa el entorno virtual:**

   - En **Windows**:
     ```bash
     .\venv\Scripts\activate
     ```

   - En **Linux/macOS**:
     ```bash
     source venv/bin/activate
     ```

2. **Ejecuta la aplicación principal (por ejemplo `app.py`):**
   ```bash
   python app.py
   ```

   Asegúrate de estar en la carpeta correcta donde se encuentra el archivo.
